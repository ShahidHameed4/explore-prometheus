# Explore Prometheus

A learning repository for understanding logging and monitoring using Prometheus.

## Overview

This repository contains experimental projects and examples for:
- Prometheus metrics collection
- Grafana dashboards
- Alerting rules
- Various exporters
- Integration with different applications

## Getting Started

Projects will be organized in subdirectories with their own documentation.

## Prerequisites

- Docker & Docker Compose
- Basic understanding of metrics and monitoring concepts
